# CLAUDE.md

## Repo structure

Each script gets its own folder with a brief `README.md`:

```
script-name/
  README.md           — what it does, setup instructions
  script-name.sh      — the script itself
  (any other files)
```

When adding a new script, always create a folder for it and include a README.
Update the table in the root `README.md` with a row for the new folder.
